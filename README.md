# 心理学研究所 ― 第8実験室 ―

**Psychology Escape**

**Version 1.0**

心理学用語を学ぶためのHTML脱出ゲームです。

---

## ゲーム概要

あなたは「心理学研究所」に閉じ込められました。

各実験室の心理学問題を解きながら、
第8実験室の脱出を目指します。

### ルール

- 外からドアを開けて実験室へ入る
- 問題に回答する
- 部屋の中からドアを開ける
- 正解なら次の実験室
- 不正解なら前の実験室
- 正解・不正解は表示されません
- 3回間違えるとGAME OVER
- GAME OVER後は間違えた問題だけ復習できます（答えは表示されません）
- 第8実験室をクリアすると修了認定証が発行されます
