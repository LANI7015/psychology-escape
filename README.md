## 音声ファイル

`sounds/` に以下を入れてください。

- door.mp3
- locked.mp3
- clear.mp3

音声がなくても、ブラウザ内蔵の簡易音で代替されます。

## 問題の編集

問題文・正解・ヒントは `data.js` で編集します。

背景画像には問題文を入れず、`question-room.png` は白いボードのある部屋の背景だけにしてください。
問題文・問題画像・入力欄・決定ボタンはHTMLで重ねて表示されます。
